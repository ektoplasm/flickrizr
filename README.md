# flickrizr
Responsive Simple Flickr Gallery

#Configuration
Install path: <root>/assets/js/flickrizr
* Remember to add your API Key and API Secret in the config file (<installation-path>/class/FlickrizrConfig.php)

#Requirements
- jQuery 1.11
- jCarousel 0.3.3
